JavaScript Practice Projects

A collection of practice projects and code snippets built with **HTML**, **CSS**, and **JavaScript**.  
This repository is designed to strengthen core web development skills through hands-on examples.

---

🚀 Features
- **HTML** → Structuring pages with semantic elements  
- **CSS** → Styling layouts, colors, and responsive design  
- **JavaScript** → Functions, arrays, DOM manipulation, events, and more  

📂 Project Structure

javascript-prac/
│── index.html        # Entry point for demos
│── style.css         # Common styles
│── script.js         # JavaScript logic
│── projects/         # Individual practice projects
│    ├── project1/
│    ├── project2/
│    └── ...
```


🛠️ Getting Started
1. Clone the repository:
   ```bash
   git clone https://github.com/R-sathaiyan/javascript-prac.git
   ```
2. Navigate into the folder:
   ```bash
   cd javascript-prac
   ```
3. Open `index.html` in your browser to explore the projects.

---

📖 Learning Goals
- Practice **JavaScript fundamentals** (functions, arrays, loops, events).
- Build confidence with **DOM manipulation**.
- Strengthen **HTML/CSS basics** for responsive layouts.
- Create small, modular projects to reinforce concepts.
